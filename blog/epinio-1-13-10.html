Announcing Epinio 1.13.10!

April 10, 2026
The Epinio Team
Epinio Maintainers

Customers currently on a prior release are encouraged to review the documentation for updates and breaking changes. You can also schedule time with us to review these updates.

New features and improvements​

  • Added additional role options and default roles.
  • Several loading-time performance improvements.
  • UI
    • Rancher Extension Epinio one-click install.
    • Added a Cluster Metrics Report download feature under System → About.

Maintenance and fixes​

  • Replaced MinIO with SeaweedFS.
  • Fixed several form updating and saving issues under certain conditions.
  • Fixed service bindings being wiped out when an application is redeployed.
  • Switched the Dex version for full OIDC support.
  • Bumped the default timeout limit for various long-running tasks.
  • Replaced SPDY with Kubernetes-native websockets.
  • Dependency updates.

For the complete technical changelog, see the 1.13.10 release on GitHub.
