Customers currently on a prior release are encouraged to review the [breaking changes](https://docs.epinio.io/reference/upgrading) documentation. You can also schedule time with us to review these updates.

<!-- truncate -->

## Helm chart enhancements

- Updated Bitnami packages.
- Node selector support for the Helm controller.
- ServiceAccount annotation support for the Epinio server.
- Additional deployment configuration options for the Epinio UI.

## Performance enhancements

- New centralized logger (we'll migrate fully to this in 1.13.6).
- Service bindings, especially multiple bindings, now run in parallel.
- Server support for tailing and date-time filtering of logs (UI support coming in 1.13.6).

## UI and CLI changes

- The new user experience has been designed, and we've begun implementing components. This reforms how the UI looks (while staying familiar) and improves performance over the next few updates up to 1.14.0.
- Dark/light mode toggle.
- Updated table component using our new design system, including client-side pagination (server-side pagination coming in 1.13.6).
- CLI logging improvements and general bug fixes.
- Several bug fixes around buttons and form flow.

## Dependency and security updates

This release also includes a range of dependency and security updates.
