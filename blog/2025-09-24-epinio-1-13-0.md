## Highlights

- Storage configurability for staging via Helm charts.
- Staging scheduling configurability via Helm charts.
- Dependency updates for the Epinio server and Epinio UI.
- Epinio UI fixes and cleanup for Rancher 2.12.0+.
- Documentation updates for the storage and scheduling capabilities.
