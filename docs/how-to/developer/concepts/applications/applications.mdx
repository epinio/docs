This guide explores working with applications. Applications are deployable workloads that Epinio builds, deploys, and manages on Kubernetes. For more information, see the [applications](../../../../reference/concepts/supported_applications.md) reference page.

## View Applications

<Tabs>
<TabItem value="ui" label="UI" default>

To view all your applications, navigate to the **Applications** page in the side navigation menu. Applications are grouped by [namespace](../namespaces/namespaces.mdx).

<img
  src={require('./apps-screen.png').default}
  alt="View of the Applications screen"
/>

</TabItem>
<TabItem value="cli" label="CLI">

List all applications:

```console
epinio app list
```

</TabItem>
</Tabs>

## Create an Application

<Tabs>
<TabItem value="ui" label="UI" default>

Click **Create** on the Applications page to open the creation modal. App creation involves four steps.

#### Application Source

Provide your application source to Epinio. If you have an [application manifest](../../../../reference/concepts/manifests.md) you can upload it directly. Otherwise, you can provide your source in any of the following ways:

- Container image
- Archive file
- From folder
- Git URL
- GitHub
- GitLab

After providing your source, select the [application chart](../app-charts/app-charts.mdx) and builder image you would like Epinio to use.

<img
  src={require('./apps-create-modal.png').default}
  alt="Create Application Modal"
  width="800"
/>

#### Application Details

Assign your application to a [namespace](../namespaces/namespaces.mdx) and provide it a name. You can also configure the number of instances, set [custom routes](../../../../reference/concepts/custom_routes.md), define environment variables, and, depending on the selected [application chart](../app-charts/app-charts.mdx), configure additional chart-specific settings.

<img
  src={require('./apps-create-details.png').default}
  alt="Create Application Modal Details"
  width="800"
/>

#### Application Bindings

Bind any [configurations](../configurations/configurations.mdx) and [services](../services/services.mdx) that your application requires. Bindings are limited to resources in the same namespace.

<img
  src={require('./apps-create-bindings.png').default}
  alt="Create Application Modal Bindings"
  width="800"
/>

#### Application Progress

Epinio will now build and deploy your application. You can watch the progress in the modal or close it, creation continues in the background and you will be notified when complete. A shell terminal will also appear at the bottom of the screen to let you monitor the process in real time.

<img
  src={require('./apps-create-progress.png').default}
  alt="Create Application Modal Progress"
  width="800"
/>

</TabItem>
<TabItem value="cli" label="CLI">

Deploy an application from the current directory:

```console
epinio app push -n myapp
```

</TabItem>
</Tabs>

## View Application Details

<Tabs>
<TabItem value="ui" label="UI" default>

Click an application name in the table to open its details page. From here you can change the number of instances, view bound services and configurations, and, if your source is from Git, change the commit to trigger a rebuild.

<img
  src={require('./apps-details-screen.png').default}
  alt="View of the Application Details screen"
/>

</TabItem>
<TabItem value="cli" label="CLI">

Show details for an application:

```console
epinio app show myapp
```

</TabItem>
</Tabs>

## Edit an Application

<Tabs>
<TabItem value="ui" label="UI" default>

To edit an application, open the action menu on the table row or details page and select **Edit**. Changes to the application source will require a rebuild and redeployment.

<img
  src={require('./apps-edit-modal.png').default}
  alt="Edit Application Modal"
  width="800"
/>

</TabItem>
<TabItem value="cli" label="CLI">

Update an application:

```console
epinio app update myapp
```

</TabItem>
</Tabs>

## Additional Actions

### App Shell

<Tabs>
<TabItem value="ui" label="UI" default>

Selecting **App Shell** in the action menu opens an interactive terminal session inside the running application container for debugging, inspection, and troubleshooting.

</TabItem>
<TabItem value="cli" label="CLI">

Open a shell session inside a running application container:

```console
epinio app exec myapp
```

</TabItem>
</Tabs>

### App Logs

<Tabs>
<TabItem value="ui" label="UI" default>

Selecting **App Logs** in the action menu opens a terminal displaying the application's runtime output, including informational messages, warnings, errors, and other diagnostic information.

</TabItem>
<TabItem value="cli" label="CLI">

Stream an application's runtime logs:

```console
epinio app logs myapp
```

</TabItem>
</Tabs>

### Last Build Logs

<Tabs>
<TabItem value="ui" label="UI" default>

Selecting **Last Build Logs** in the action menu opens a terminal displaying logs from the latest application build.

</TabItem>
<TabItem value="cli" label="CLI">

Stream logs from the latest application build:

```console
epinio app logs myapp --staging
```

</TabItem>
</Tabs>

### Rebuild

<Tabs>
<TabItem value="ui" label="UI" default>

Selecting **Rebuild** in the action menu creates a new application image from the application's source code and deployment configuration, then deploys the updated version.

</TabItem>
<TabItem value="cli" label="CLI">

Rebuild and redeploy an application:

```console
epinio app restage myapp
```

</TabItem>
</Tabs>

### Restart

<Tabs>
<TabItem value="ui" label="UI" default>

Selecting **Restart** in the action menu restarts an application's running instances without rebuilding its container image or changing its configuration.

</TabItem>
<TabItem value="cli" label="CLI">

Restart an application's running instances:

```console
epinio app restart myapp
```

</TabItem>
</Tabs>

### Export

<Tabs>
<TabItem value="ui" label="UI" default>

Selecting **Export** in the action menu downloads an application's deployment artifacts — including its manifest, application charts, and container images — for backup, sharing, or importing into another Epinio environment.

<img
  src={require('./apps-export-modal.png').default}
  alt="Export App Modal"
  width="500"
/>

</TabItem>
<TabItem value="cli" label="CLI">

Export an application's artifacts:

```console
epinio app export myapp
```

</TabItem>
</Tabs>
