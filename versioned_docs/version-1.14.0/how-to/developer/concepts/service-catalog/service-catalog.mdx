This guide explores working with the service catalog. Epinio provides a catalog of templates that are used to create [service instances](../services/services.mdx). For more information see the [service catalog reference](../../../../reference/concepts/catalog.md) page.

## View Service Catalog

<Tabs>
<TabItem value="ui" label="UI" default>

To view the service catalog, navigate to the **Catalog** page under the **Services** section of the side navigation menu.

<img
  src={require('./catalog-screen.png').default}
  alt="View of the Services Catalog screen"
/>

</TabItem>
<TabItem value="cli" label="CLI">

List all available catalog services:

```console
epinio service catalog
```

</TabItem>
</Tabs>

## View Catalog Service Details

<Tabs>
<TabItem value="ui" label="UI" default>

Click on a service catalog card to open its details page. Here you can view all instances created from that service, create new instances, and edit or delete existing ones. See the [services how-to](../services/services.mdx) for more information.

<img
  src={require('./catalog-service-screen.png').default}
  alt="View of the Catalog Service Details screen"
/>

</TabItem>
<TabItem value="cli" label="CLI">

Show details for a specific catalog service:

```console
epinio service catalog myservice
```

</TabItem>
</Tabs>
