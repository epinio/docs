Epinio welcomes your participation in the project, via GitHub issues and pull
requests. Use the guides below for the details of each contribution track.

## Contribution guides

Pick the track that fits your contribution:

- **Documentation** — improve or add docs following our conventions. See the [Documentation Guide](./documentation-guide.md).
- **Development** — set up a local environment and build Epinio. Start with the [Prerequisites](./development/prereqs.md), then the [Platform](./development/platform.md) and [UI](./development/ui/standalone.md) guides, and review the [Development Standards](./development/dev-standards.md).

## Reporting issues

Found a bug or have a feature request? Open an issue in the repository it belongs
to, and follow that repository's issue template:

- Core and server: [epinio/epinio](https://github.com/epinio/epinio/issues/new/choose)
- UI and dashboard: [epinio/ui](https://github.com/epinio/ui/issues/new/choose)
- Documentation: [epinio/docs](https://github.com/epinio/docs/issues/new/choose)
- Helm charts: [epinio/helm-charts](https://github.com/epinio/helm-charts/issues/new/choose)
